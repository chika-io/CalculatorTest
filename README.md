# 🧮 Calculator CLI

A clean, interactive command-line calculator built in Python — supporting both basic and advanced mathematical operations.

> Built by [Ukpai Chika Nwakaibeya](https://github.com/chika-io)

---

## Features

- Basic operations: Addition, Subtraction, Multiplication, Division, Modulus, Power
- Advanced operations: Square Root, Logarithm (base 10), Natural Logarithm
- Calculation history — view all past calculations in a session
- Input validation and helpful error messages
- Clean, easy-to-navigate menu interface

---

## Getting Started

### Prerequisites
- Python 3.x installed on your machine

### Run the App

```bash
# Clone the repository
git clone https://github.com/chika-io/calculator-cli.git

# Navigate into the folder
cd calculator-cli

# Run the calculator
python3 calculator.py
```

---

## Usage

Once running, you'll see a menu like this:

```
=============================================
       CALCULATOR CLI  —  chika-io
=============================================

  BASIC OPERATIONS
  ─────────────────
  1. Addition           ( a + b )
  2. Subtraction        ( a - b )
  3. Multiplication     ( a × b )
  4. Division           ( a ÷ b )
  5. Modulus            ( a % b )
  6. Power              ( a ^ b )

  ADVANCED OPERATIONS
  ─────────────────────
  7. Square Root        ( √a )
  8. Logarithm base 10  ( log a )
  9. Natural Logarithm  ( ln a )

  ─────────────────────
  H. History
  C. Clear Screen
  Q. Quit
```

Simply enter the number of the operation you want, then follow the prompts.
